# AI Resume Analyzer

This project analyzes a resume and compares it with a job description using NLP techniques.

## Features
- Resume PDF parsing
- Skill detection
- Job description matching
- Resume score calculation
- Missing skills detection
- AI suggestions

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLP (TF-IDF)

## How to Run

pip install -r requirements.txt
streamlit run app.py
